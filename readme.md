The repository contains the sample code for a Simple Todo List App to support a medium article.
The article is published at: https://medium.com/@iakhilesh/api-development-using-python-and-fast-api-069a71db4aae

## Initial Setup
Make sure python is installed on your machine. Most of the machines nowadays come pre-installed specially MACs
### Python Installation
```
brew install python
```
```
python3 --version
```
### PIP Installation
```
pip3 --version
```
## Install Dependencies
```
pip install -r requirements.txt
```
## Start Server
```
uvicorn main:app --reload
```
